# SpeechTranslationAzure
Testing Out Microsoft Azure's Speech Recognition / Translation!

### Instructions
1. Set your Azure Speech Subscription Key inside `web/subscriptionKey.js`
2. Run `node index.js`
3. Open `http://localhost:8100/web` in a Browser
